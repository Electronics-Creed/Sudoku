# Sudoku

In this project a program is built to solve sudoku by scanning an image of the puzzle. We have used backtracking algorithm to solve the puzzle. We have used neural networks and edge detection methods to scan the sudoku.

## How To Use

 - Run the file sudoku(no opencv).py to play the game in console. 
 - Run the file final_sudoku.py to play the game in an interactive window created using OpenCV.
